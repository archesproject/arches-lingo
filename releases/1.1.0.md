### Installation

1. Install the Python package:
    ```
    pip install arches-lingo
    ```

2. Add the required applications to `INSTALLED_APPS` in your project settings, below the name of your project:
    ```python
    INSTALLED_APPS = (
        ...
        "my_project_name",
        "arches_querysets",
        "arches_component_lab",
        "arches_controlled_lists",
        "arches_lingo",
        "pgtrigger",
    )
    ```

3. Add the required settings to your project's `settings.py`. See the [README](https://github.com/archesproject/arches-lingo) for the full configuration reference.

4. Run migrations:
    ```
    python manage.py migrate
    ```

5. Build the frontend:
    ```
    npm install
    npm run build_development
    python manage.py collectstatic
    ```

6. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
